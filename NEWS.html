NEWS

Major changes in release 1.0:

* ebnetd has a table of file descriptor cache now.
* ebhttpd is more robust against HTTP request with corrupt header.

Major changes in release 1.0beta0:

* ndtpd no longer support `alias-eiwa', `alias-waei' or `alias-kojien'.
* The defulat configuration file for ndtpd and ebhttpd is `ebnetd.conf',
  not `ndtpd.conf'.
